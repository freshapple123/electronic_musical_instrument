## 🎵 기능 소개

- **피아노 모드** : 터치 시 해당 음이 출력되는 가상 피아노  
- **전자 드럼 모드** : 다양한 드럼 소리를 지원하는 드럼 인터페이스  
- **MP3 모드** : 미리 설정된 배경 음악 재생 기능  
- **녹음 기능** : 사용자가 연주한 내용을 녹음 가능  
- **녹음 듣기** : 녹음된 파일을 재생하여 다시 들어볼 수 있음
